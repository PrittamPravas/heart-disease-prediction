# Heart Disease Prediction & Clinical Analytics

## Project Overview

This project predicts the likelihood of heart disease using machine learning techniques and clinical patient data. The objective is to identify key risk factors and support early diagnosis through predictive analytics.

## Dataset

* 1,025 patient records
* 13 clinical features
* Includes age, cholesterol, chest pain type, ECG results, maximum heart rate, thalassemia type, and exercise-induced angina

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Power BI
* Jupyter Notebook

## Models Evaluated

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)

## Best Model

### Support Vector Machine (SVM)

Performance:

* Accuracy: 99%
* Precision: 100%
* Recall: 98%

## Data Analysis

Performed:

* Exploratory Data Analysis (EDA)
* Correlation Analysis
* Feature Engineering
* Data Cleaning
* Box-Cox Transformation
* Model Evaluation

## Key Insights

The most influential predictors of heart disease were:

* Chest Pain Type
* Thalassemia Type
* Number of Major Vessels
* Exercise-Induced Angina

## Dashboard

A Power BI dashboard was developed to visualize:

* Patient demographics
* Risk factors
* Model performance metrics
* Clinical insights
